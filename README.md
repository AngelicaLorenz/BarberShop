# 💈 BarberShop

Site institucional de uma barbearia desenvolvido como projeto de prática em **Desenvolvimento Front-End**.

O projeto tem como objetivo aplicar conceitos fundamentais de **HTML, CSS e JavaScript**, criando uma página web organizada, estilizada e com interatividade básica.

## 🎯 Objetivo

Praticar habilidades essenciais do desenvolvimento web, como:

- estruturação de páginas com HTML
- estilização com CSS
- interatividade com JavaScript
- publicação de projetos com GitHub Pages
